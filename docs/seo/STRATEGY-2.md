# 《本站 SEO 作战方案》(第二轮 · 策略主场)

完成日期：2026-06-18 ｜ 唯一成败指标：90 天 GSC 曝光

## 定位（一页）
- **是什么**：AI 工具「决策站」——全网唯一把每个工具的**真实缺点、真实免费额度、真实花费**摆上台面的地方。不是"又一个大全"。
- **服务谁**：处在"为某个具体任务掏钱/做决定"那一刻的人，在 2–3 个工具间纠结、怕被联盟软文带偏。
- **决策词**：`best X for 人群`、`X vs Y`、`X alternatives`、`is X worth it`、`X free tier limits`、`cheapest X`、`X without 水印/注册`、`how to cancel X`。
- **缝隙**：竞品全是**发现型数据**(名/标签/流量/星级)；**决策型数据没人做**——因为联盟站不敢写缺点、厂商不黑自己。结构性、抄不动。

## 数据普查（护城河来源）
140 工具 / 13 类。**notFor 诚实缺点 100%**、免费层 72%(101)、带$价格 91%(127)、bestFor 100%。13/13 类 ≥4 工具，12/13 类 ≥4 免费。→ 决策数据完整，是结构性护城河。

## 机会地图（SERP 弱点 → 缝隙）
| 词族 | 首页现状 | 机会 |
|---|---|---|
| `ai tools pricing comparison` | GitHub README 排进首页 | 极高 |
| `is X worth it` | 实体歧义 + 论坛 | 极高 |
| `X free tier limits` / `best free X` | 全网只给"Free"角标，无真实额度 | 极高 |
| `best ai tools for 人群/faceless youtube` | 全是利益相关厂商 + 联盟 | 高 |
| `X without 水印/注册` | 只有产品落地页 | 高 |
| `how to cancel X` | 泛"取消"聚合站 | 中高 |

## 选定奇招（5，其中 4 个清单外原创）
1. ⭐ **诚实反向层**（notFor 100%）：详情页"值不值"判决 + `is X worth it`。抄不走：联盟站不敢写缺点。
2. ⭐ **真实免费额度规范化索引**：工具 | 免费字数/额度/图数 | 水印 | 信用卡。GEO 引用金矿。**需 Anthropic key 富集**（见交接文档#2）。
3. **按产出算钱**（扩展定价报告）：每千字/每分钟/每百图的真实成本；击败 GitHub README。
4. ⭐ **最省钱 AI 工具栈生成器**：选目标→最省跨类目组合+月总价+免费版。抄不走：目录都是一卡一工具。
5. ⭐ **逃离/迁移意图页**：替代+免费+取消。吃竞品不碰的售后意图。

## 页面架构（本轮已落地，见交接文档#2）
- 核心：`/match` 选型器（首选功能）。
- 程序化：`/best/[task]/cheapest`、`/stacks/[outcome]`、详情"值不值"+最后更新。
- 深度内容：`/guides` 4 篇手写 + `/about` + `/contact`。

## GEO
答案前置 + TL;DR + FAQ schema + 对齐字段对比表 + 带来源 + "最后更新"；大量 best/榜单/对比/缺点页。实测：原始数据 + 有界事实 + 对齐表格最易被 ChatGPT/Perplexity 引用。
